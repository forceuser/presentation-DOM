Поиск по дереву DOM

  • element.getElementById(elementId) - поиск элемента по его id
  • element.getElementsByName(name) - поиск элементов по аттрибуту name, возвращает живую колекцию
  • element.getElementsByTagName(tagName) - поиск элементов по имени тега
  • element.getElementыByClassName(className) - поиск элементов по имени класса
  • element.querySelector(selector) - поиск первого элемента который соответсвует css селектору
  • element.querySelectorAll(selector) - находит коллекцию элемента соответсвующих css селектору
  • element.closest(selector) - находит ближайшего родителя либо себя если соответсвует селектору
  • element.matches(selector) - проверяет соостветствует ли элемент указанному селектору
