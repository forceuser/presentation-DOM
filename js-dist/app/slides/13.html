Обьекты, свойства, дескрипторы

  • Object.create(proto[, propertiesObject]) - создает который наследуется от указанного, необязательный параметр propertiesObject обьявляет свойтва обьекта в виде дескрипторов
  • Object.defineProperty(obj, prop, descriptor) - устанавливает дескриптор descriptor свойтва prop обьекта obj
  • Object.defineProperties(obj, props) - для обьекта obj устанавливает дескрипторы для свойств указанных в обьекте props (ключ - имя свойства, значение- дескриптор)

Дескрипторы

  • configurable - если false то свойство не может быть удалено и не может быть модифицырован дескриптор
  • enumerable - если true то свойство доступно для перечисления (например через цикл for..in)
  • value - устанавливает значение свойства
  • writable - если свойство доступно для записи при помощи оператора присваивания
  • get - геттер, функция возвращаемое значение которой будет использоватся как значение свойства
  • set - сеттер (function(value){}), функция вызывается при присвоении свойству нового значения
    геттер и сеттер не могут быть использованы одновременно с десрипторами value и writable
