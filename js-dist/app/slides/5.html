Типы узлов HTML DOM (свойство node.nodeType)

  • ELEMENT_NODE (nodeType = 1) - узлы с типом элемент, соответсвуют html тегам
  • TEXT_NODE (nodeType = 3) - узлы с текстовой информацией
  • COMMENT_NODE (nodeType = 8) - тип узлов комментариев
  • DOCUMENT_NODE (nodeType = 9) - тип корневого узла документа
  • DOCUMENT_TYPE_NODE (nodeType = 10) - узел определяющий DOCTYPE
  • DOCUMENT_FRAGMENT_NODE (nodeType = 11) - фрагмент документа

Создание DOM элементов

  • document.createElement(tagName) - создание элемента с указанным тегом
  • document.createComment(text) - создание комментария
  • document.createTextNode(text) - создание текстового узла
  • document.createFragment() - создание фрагмента документа
