Навигация по дереву DOM (по всем узлам)

  • element.childNodes[] - коллекция дочерних узлов
  • element.parentNode - ссылка на родительский узел (для узла document == null )
  • element.firstChild - ссылка на первый дочерний узел
  • element.lastChild - ссылка на последний дочерний узел
  • element.nextSibling - ссылка на следующий соседний узел
  • element.previousSibling - ссылка на предыдущий соседний узел
