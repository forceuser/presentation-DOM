Манипуляция с узлами DOM

  • parent.appendChild(child) - добавление дочернего узла к текущему
  • parent.removeChild(child) - удаление дочернего узла
  • parent.replaceChild(child, oldChild) - заменяет дочерний элемент oldChild на child, возвращает oldChild
  • parent.insertBefore(child, referenceElement) - добавляет дочерний элемент child перед дочерним элементом referenceElement
  • parent.insetAdjacentHTML(position, text) - добавляет HTML содержимое в указанное место position ('beforebegin','afterbegin','beforeend','afterend')

Атрибуты DOM элементов

  • element.attributes[] - ссылка на коллекцию атрибутов элемента
  • element.addAttribute(name, value) - добавляет атрибут name и устанавливает ему значение value
  • element.getAttribute(name) - получает значение атрибута с указанным именем
  • element.removeAttribute(name) - удаляет атрибут с указанным именем
  • element.hasAttribute(name) - проверяет содержится ли указанный атрибут в коллекции
