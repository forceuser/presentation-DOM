Навигация по дереву DOM (только элементы)

  • element.children[] - коллекция дочерних элементов
  • element.parentElement - ссылка на родительский элемент (для элемента html == null )
  • element.firstElementChild - ссылка на первый дочерний элемент
  • element.lastElementChild - ссылка на последний дочерний элемент
  • element.nextElementSibling - ссылка на следующий соседний элемент
  • element.previousElementSibling - ссылка на предыдущий соседний элемент
