Определение относительного положения в дереве

  • node.contains(otherNode) - проверяет является ли node потомком otherNode
  • node.compareDocumentPosition(otherNode) - сравнивает положение в документе node относительно otherNode, возвращает битовую маску

Значения битовой маски

  • DOCUMENT_POSITION_DISCONNECTED = 1 - node исключен из документа
  • DOCUMENT_POSITION_PRECEDING = 2 - node предшествует otherNode
  • DOCUMENT_POSITION_FOLLOWING = 4 - node следует после otherNode
  • DOCUMENT_POSITION_CONTAINS = 8 - node содержит otherNode
  • DOCUMENT_POSITION_CONTAINED_BY = 16 - otherNode содержит node
