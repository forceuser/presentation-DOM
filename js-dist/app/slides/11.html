События

  • element.addEventListener(имя-события,callback,capture=false) - добавление обработчика
  • element.removeEventListener(имя-события,callback,capture=false) - удаление обработчика
Пример наглядно демонстрирующий фазы обработки событий

Обьект события

  • event.preventDefault() - предотвращает действие по умолчанию для текущего события
  • event.stopPropagation() - останавливает передачу события по дереву
  • event.stopImmediatePropagation() - останавливает дальнейшую передачу события в другие обработчики для текущего эл-та
